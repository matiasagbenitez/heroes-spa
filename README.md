# Heroes SPA

This is a React project built with Vite, featuring a single-page application (SPA) of heroes. Its purpose is to review basic concepts of routing and context, incorporating public and private routes, and also query-strings for search. It also includes testing using react-testing-library, bootstrap styles, and animations.

## Installation

To install the project's dependencies, make sure you have Node.js and NPM installed on your machine. Then, follow these steps:

1. Clone this repository to your local machine
2. Open a terminal and navigate to the project directory
3. Run the command `npm install` to install the dependencies

## Usage

Once you have installed the dependencies, you can run the server locally. To do this, run the following command in the terminal:

`npm run dev`

This will start the server and open a browser window with the application.
